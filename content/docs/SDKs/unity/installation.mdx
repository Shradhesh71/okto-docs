# Installation and Setup for Unity SDK

Follow this guide to get started with the Unity SDK and integrate it into your project seamlessly.

---

## Step 1: Prerequisites

Before starting, ensure you have the following:

1. **Unity Installed**  
   - Download and install the latest version of Unity from [Unity Hub](https://unity.com/download).
   - Ensure your Unity version is compatible with the SDK.

2. **.NET Framework**  
   - The Unity SDK requires .NET Framework 4.6 or higher. Check your project's `.NET API Compatibility Level` under `Player Settings > Other Settings`.

3. **Access to the SDK**  
   - Obtain the SDK package from the official website or GitHub repository.

---
